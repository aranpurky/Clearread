# Clearread

**Clearread** is a modern, distraction-free reading tool designed to help you process messy text, articles, or essays faster.

It runs entirely in a single HTML file‚Äîno servers, no installations, and no dependencies required.

## ‚ú® Features

* **Smart Highlighting**: Uses a statistical density algorithm to identify and highlight the top 20% most relevant sentences based on keyword frequency.
* **Distraction-Free UI**: A clean, glassmorphism-inspired interface that puts your content center stage.
* **Adaptive Themes**: Switch instantly between Modern (White), Paper (Cream), Graphite, and Midnight modes to suit your lighting.
* **Insight Stats**: Get real-time metrics on:
    * Word Count
    * Reading Time (based on 220 wpm)
    * Sentence Count
    * Text Complexity (% of long words)
    * Information Density
* **Privacy First**: All processing happens locally in your browser. Your text never leaves your device.
